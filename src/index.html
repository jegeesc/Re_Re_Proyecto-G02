INFÓRMATE

¿Qué sensores ofrecemos?

Sensor de temperatura

El sensor evalúa, mide y recopila la temperatura ambiental, ajustando el tiempo entre cada medición. Así los agricultores pueden determinar y variar el trabajo que realizan según las condiciones en las que se encuentre la sonda, sin tener que estar pendiente de la plantación en cada momento.

Sensor de humedad

El sensor está diseñado para realizar un control a tiempo real sobre la humedad y estado de las plantas, permitiendo así a los agricultores regular el riego de manera más eficiente. Siendo avisados de los diferentes cambios que se puedan encontrar en la plantación, sin necesidad de que alguien esté en el lugar del cultivo.

Sensor de salinidad

Este sensor está diseñado para controlar el nivel de salinidad de los pozos de riego de las diferentes plantaciones. Ayudando a los agricultores a saber qué pozos pueden ser utilizados en cada momento, teniendo la facilidad de comprobarlo al alcance de tu mano, y además mejorar la producción.

Sensor de iluminación

El sensor está diseñado para controlar los niveles de luminosidad, y así determinar si la sonda se encuentra en un ambiente de luz artificial, luz natural, incluso si es de noche o la luz está apagada. De esta manera se mejora el rendimiento de los invernaderos y al mismo tiempo se sabe las mejores condiciones de crecimiento de los cultivos.

Acelerómetro

El sensor está diseñado para detectar la aceleración puntual sufrida en cada uno de los ejes del mismo. Estos cambios de aceleración proporcionan al sensor información acerca del movimiento que se está describiendo y que le afectan en función del tiempo.


¿Cómo visualizo los datos?

Gráfica circular

Este tipo de representación está diseñada para poder ver con exactitud los datos en función del tiempo. Puediendo asi,ver como reaccionan a diferentes tratamientos o temperaturas y compararlo con resultados obtenidos con anterioridad.

Gráfica lineal

Este otro tipo de represents está diseñada para ver el porcentaje que nos está midiendo cada sonda pudiendo así ver como reaccionan a diferentes tratamientos o temperaturas y compararlo con resultados anteriores.

PREGUNTAS FRECUENTES

¿Cómo se instalan las sondas?

Nuestras sondas son muy sencillas de instalar, solo hace falta comprar las unidades de sondas que desees y un técnico las enlazará con su usuario.

¿Hay límite de sondas?

No hay ningún límite de sondas, nuestro software se encarga de organizar tantas sondas como necesites al alcance de un click.

¿Són fáciles de reparar?

Nuestras sondas están pensadas para poder ser modulables, así que si algún sensor se estropea, puede ser cambiado por un precio económico.
